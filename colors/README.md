#COlors Preview

To apply the colors:

- Git clone this repo
- add this code to .xresources
`#include <path to colors>`
eg:
   `#include </home/deadsec/.termscolors/colors1.xres>`
- Update the xresources
`xrdb /home/deadsec/.Xresources`
